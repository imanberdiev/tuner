# 🚀 Инструкция: Получить APK через GitHub (БЕСПЛАТНО, без установки ПО)

---

## Шаг 1 — Создайте аккаунт GitHub
Если у вас его нет: https://github.com/signup (бесплатно)

---

## Шаг 2 — Создайте новый репозиторий
1. Зайдите на https://github.com/new
2. Название: `tuneup-android`
3. Выберите **Private** (или Public)
4. Нажмите **Create repository**

---

## Шаг 3 — Загрузите файлы проекта

На странице созданного репозитория нажмите:
**"uploading an existing file"** (или Add file → Upload files)

Загрузите ВСЕ файлы и папки из архива `TuneUp-Android-Project.zip`.
Структура должна быть такой:
```
(корень репозитория)
├── .github/
│   └── workflows/
│       └── build.yml        ← ВАЖНО! Должен быть здесь
├── app/
│   └── src/...
├── gradle/
│   └── wrapper/
│       └── gradle-wrapper.properties
├── gradlew
├── build.gradle
└── settings.gradle
```

> 💡 Совет: Распакуйте ZIP, зайдите в папку `android-project` и загружайте файлы оттуда.

Нажмите **Commit changes**

---

## Шаг 4 — Запустите сборку APK

1. В репозитории нажмите вкладку **Actions** (сверху)
2. Слева выберите **"Build TuneUp APK"**
3. Нажмите кнопку **"Run workflow"** → **"Run workflow"**
4. ⏳ Подождите 3–5 минут (можно следить за прогрессом)

---

## Шаг 5 — Скачайте APK

После завершения (зелёная галочка ✅):
1. Нажмите на завершённый workflow
2. Прокрутите вниз до раздела **Artifacts**
3. Нажмите **TuneUp-v1.0-debug** → скачается ZIP с APK внутри

---

## Шаг 6 — Установите на телефон

1. Скопируйте `app-debug.apk` на телефон (через USB, Telegram, email)
2. На телефоне откройте файловый менеджер → найдите APK
3. Нажмите на файл
4. Если появится предупреждение — нажмите **"Установить всё равно"**
   - Настройки → Безопасность → включите "Неизвестные источники"
5. Готово! Приложение **TuneUp** появится на рабочем столе 🎸

---

## ❓ Частые вопросы

**Сборка упала с ошибкой?**
→ Проверьте, что файл `gradlew` загружен в корень репозитория (не внутри подпапки)

**Не вижу вкладку Actions?**
→ Settings → Actions → General → Allow all actions → Save

**APK не устанавливается?**
→ Настройки телефона → Приложения → Специальный доступ → Установка неизвестных приложений → разрешите вашему файловому менеджеру
